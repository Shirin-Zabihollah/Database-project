# Database-project
University Database project 
Symptom Tracker — Full Database-Driven Web Application

A complete medical tracking system built with PHP & MySQL.

 **Summary**

This project is a fully functional backend-driven web application designed to manage patients, doctors, medical visits, symptoms, diagnoses, and prescriptions.
It demonstrates strong skills in database design, data modeling, and backend development.

 **What I Built**
 
 **Backend (PHP + MySQL)**
 
CRUD operations for all entities
Form handling & server-side validation
Foreign key constraints & relational integrity
Clean procedural PHP structure

 **Database Design**
 
Professional ER Diagram (McGraw-Hill notation)
Normalized relational schema
6 fully relational tables with strong FK links

 **Frontend (HTML)**
 
Structured forms for all operations
Clean, simple user flow




| Technology         | Usage                           |
| ------------------ | ------------------------------- |
| **PHP**            | Backend logic & data processing |
| **MySQL**          | Database + constraints          |
| **XAMPP / Apache** | Local hosting                   |
| **HTML**           | Forms & UI                      |




**Main Features**

Register & list patients
Register doctors
Add visits & diagnoses
Log symptoms
Add prescriptions
View reports for all data
